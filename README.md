## Usage

### Prefix Search

```go
import "github.com/joway/trie"

dict := map[string]interface{}{
		"/abc": "2",
		"/a":   "1",
		"/ac":  "3",
		"/b":   "4",
		"/bc":  "5",
		"/bca": "6",
		"/ba":  "7",
		"/cba": "8",
}

tree := trie.Build(dict)

prefix, val := tree.PrefixSearchString("/a")
```

### Prettify Output

```go
output := tree.Prettify()
fmt.Println(output)
```

```
*->/->a->b->c
||  |  ->c
||  ->b->a
||  |  ->c->a
||  ->c->b->a
```
